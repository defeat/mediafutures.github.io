INFORMATION RETRIEVAL

Information Retrieval is the act of obtaining information from a library of information resources.

A query is passed into the IR system and the results shown must be relevant to the query. All results are given a value which allows ranking of results to match the query and to display the closest relevant result to a particular query

We use IR systems every single day. Whether that be in University Libraries or Public libraries, in the form of book search systems; or online.

Web search engines; such as Google, Yahoo and Bing are common types of IR Systems.

Information Retrieval systems use different models to obtain the relevance ranking of a result to a query

Google, The world's most used Web Search Engine was built on the Standard Boolean Model. This model passes a query as a set of terms throughout all the data they have.

The terms are matched against found documents and documents with the most terms are ranked highest.

Standard Boolean Indexing Model

Instead of linearly searching through each document with terms, A matrix of terms is created in advance which allows the passing of a boolean determinant to match a query.

Suppose a collection of Shakespeare's works.

An information index is created for the words in each Shakespeare play. They are passed as a 'term' in the matrix.

Depending on the column, a vector is created for a specific term which is contained within that play. This becomes a vector which has a 'true' value
